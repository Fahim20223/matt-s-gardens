# 🌿 Matt's Gardens

A modern and responsive gardening services website built with **Next.js**, **Tailwind CSS**, **DaisyUI**, and **Framer Motion**. The project focuses on delivering a clean user interface, smooth animations, and an excellent user experience across all devices.
